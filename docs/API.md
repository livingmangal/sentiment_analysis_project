# Sentiment Analysis API Documentation

This document provides details about the API endpoints used in the Sentiment Analysis application.

---
## Base URL
http://localhost:5000

## Predict Sentiment

### Endpoint

POST /predict

---

### Description

Analyzes the sentiment of input text and returns prediction results.

---

### Request

#### Headers

Content-Type: application/json

#### Body

```json
{
  "text": "This movie was fantastic!"
}
```

### Successful Response
**Status Code:** 200 OK

```json
{
  "sentiment": "Positive",
  "confidence": 0.92,
  "raw_score": 2.45
}
```

### Response Fields


| Field      | Type   | Description                                          |
| ---------- | ------ | ---------------------------------------------------- |
| sentiment  | string | Predicted sentiment label (`Positive` or `Negative`) |
| confidence | float  | Confidence score between 0 and 1                     |
| raw_score  | float  | Raw sentiment score generated by the model           |

### Error Responses

#### 400 Bad Request – Missing Text

```json
{
  "error": "Text field is required"
}
```

#### 400 Bad Request – Empty Text

```json
{
  "error": "Input text cannot be empty"
}
```

#### 400 Bad Request – Invalid Request Format

```json
{
  "error": "Invalid JSON format"
}
```

#### 500 Internal Server Error

```json
{
  "error": "Internal server error"
}
```

### Example Usage

#### cURL

```bash
curl -X POST http://localhost:5000/predict \
     -H "Content-Type: application/json" \
     -d '{"text": "I love this product!"}'
```

#### JavaScript (Fetch API)

```javascript
fetch('http://localhost:5000/predict', 
{
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    text: 'I love this product!'
  })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error(error));
```

### Notes

The API accepts only English text.

Sentiment classification is limited to Positive and Negative.

All responses are returned in JSON format.

Authentication is not required.

### Model Information

Model Type: Bidirectional LSTM

Framework: PyTorch

Confidence Score: Derived from model probability output

Output: Sentiment label with confidence score
